Bienvenue sur SOIVD

Description :

Vous pouvez accéder à plusieurs informations à la fois sur une ville en écrivant une requete SQL. Vous trouverez ci-bas (en image) toutes colonnes que vous pouvez utiliser. Nous avons quatre soucre de données (api) qui sont ensuite intégrer virtuellement dans une seule base de données.

  • API météo : Information météo par ville
  • API Energie : Information sur la consommation d'energie par ville
  • API Séisme : Information sur le risque de seisme par ville
  • API Population : Information Démographiques par ville

Notre systéme ne prend en compte que les opérations de projections et de restrictions :

SELECT col1,col2,...
FROM DF
WHERE (your restrictions)
